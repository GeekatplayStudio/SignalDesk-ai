
# LLM Quality & Compliance Dashboard (Next.js)

Next.js 14 + React 18 + Tailwind + shadcn/ui implementation of the Sentinel dashboard. Based on the provided Figma design: https://www.figma.com/design/U8bEmODauOd99A8V7KyiYY/LLM-Quality---Compliance-Dashboard.

## Running the code

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the dashboard.
