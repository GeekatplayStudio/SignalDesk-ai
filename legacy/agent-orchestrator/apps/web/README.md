
# Developer Console (Web)

This app is the admin/mission-control frontend for the MVA platform.

Design source:
- https://www.figma.com/design/bHcOZPjxy1v1DPsHHPlsxV/Agent-Developer-Console-Design

## Local run
1. `npm install`
2. `cp .env.example .env`
3. `npm run dev`

## Scripts
- `npm run dev`: start local Vite server
- `npm run build`: production build
- `npm run preview`: preview production build
